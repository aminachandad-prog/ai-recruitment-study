Repository: AI Recruitment Study

This repository contains all data and materials supporting the article:

"Transformations in Talent Acquisition: Measuring AI’s Effects on Recruitment Efficiency and Bias"

Contents:

1. survey_responses_anonymized.csv
   → Raw anonymised dataset (423 observations)

2. variable_codebook.csv
   → Variable definitions and coding structure

3. questionnaire.pdf
   → Full survey instrument

4. consent_form.pdf
   → Participant information and consent document

5. analysis_script.R
   → Statistical analysis code (EFA, regression)

All data are anonymised and shared under CC BY 4.0 licence.
